# 🧮 Simple Python Calculator
 This is a simple calculator project created using Python while       learning programming fundamentals. It performs basic mathematical    operations based on the user's choice.
# Features
  Addition (+)
  Subtraction (-)
  Multiplication (*)
  Division (/)
  Power (**)
  Technologies Used
  Python 3
# How to Run
  Download or clone this repository.
  Open the project in your Python editor.
  Run the Python file.
  Enter two numbers.
  Choose the operation.
  The program will display the result.

# Example

  Enter first number: 10
  Enter second number: 5

 1. Addition
 2. Subtraction
 3. Multiplication
 4. Division
 5. Power

  Enter your choice: 3

  Result: 50
