# 📚 Student Marks Calculator (Python)

A simple Python program that calculates a student's total marks, percentage, and grade based on marks entered for six subjects.

# 🚀 Features

- Enter student's name
- Input marks for 6 subjects
- Calculate total marks
- Calculate percentage
- Display grade automatically using "if-elif-else"
- Beginner-friendly Python project

# 🛠️ Technologies Used

- Python 3
- Basic Python Functions
- Conditional Statements ("if", "elif", "else")
- User Input ("input()")
- Arithmetic Operations

Percentage| Grade
90% and above| A+

60% - 69%| C
35% - 59%| D
Below 35%| FAIL

# ▶️ How to Run

1. Install Python 3.
2. Save the program as "student_marks.py".
3. Open a terminal in the project folder.
4. Run:

python student_marks.py

5. Enter the student's details and subject marks.
